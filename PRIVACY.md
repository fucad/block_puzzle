# Privacy Policy — Block Puzzle

**Last updated: 2026-07-12**

Block Puzzle collects **nothing**. This is the whole policy, but stores
require the details, so:

## What we collect

Nothing. The app contains:

- no analytics or telemetry of any kind,
- no advertising SDKs,
- no crash reporting services,
- no accounts, logins, or user identifiers,
- no access to contacts, location, camera, microphone, or files.

## What stays on your device

Your scores, settings, and quest progress are stored only on your
device. Deleting the app deletes them. We cannot see them — there is no
server of ours to send them to.

## The one network request

The app periodically downloads new quest levels from this public GitHub
repository via GitHub's content servers (`raw.githubusercontent.com`).
This is an anonymous file download, like visiting any website; GitHub's
standard server logs apply to it (see
[GitHub's privacy statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement)).
No data about you or your play is sent — the request contains nothing
but "please give me the level list". The game is fully playable
offline.

## Children

The app is suitable for all ages. Since it collects no data at all, it
collects no data from children.

## Changes

Any change to this policy will appear in this file's public git
history — which will forever show that nothing was ever collected.
